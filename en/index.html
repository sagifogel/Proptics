Proptics is a Profunctor Optics and Lenses library for Scala

Profunctional

Proptics uses a Profunctor encoding for the internal representation of its optics. You can learn what is profunctor in the profunctor section.

Simple to use and easy to learn

Proptics API was designed to resemble the Scala standard library.

Cats Friendly

Proptics is built on top of Cats and Spire
