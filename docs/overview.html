Overview

Proptics is a Profunctor Optics and Lenses library for Scala (and Scala.js)

Getting Started

To get started with sbt, simply add the following line to your build.sbt file.

libraryDependencies ++= Seq(
  "io.github.sagifogel" %% "proptics-core" % "0.5.1",
  "io.github.sagifogel" %% "proptics-profunctor" % "0.5.1",
  "io.github.sagifogel" %% "proptics-macros" % "0.5.1"
)

Published for Scala 2.12, 2.13, 3.2 and Scala.js. For changes, refer to the release notes.
